This is a Personal Expense Tracker Application made using Django. The source code of this project can be found on Github.

Users can create expenses based on categories and incomes based on different sources. Users can filter income/expense based on received/spent today, weekly, monthly or yearly. All these information are present on the dashboard and income summary and expense summary are available in the form of different Graphs. Users can also download this data in form of Pdf, Csv and Excel. Users can also import incomes and expenses from a CSV or an Excel File. Users also have a Profile where they can change currency to be displayed with expenses and incomes.
